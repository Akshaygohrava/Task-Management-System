## ğŸš€ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Akshaygohrava/Task-Management-System.git
   cd Task-Management-System
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open in your browser:**
   ```
   http://localhost:3000
   ```
